Built with Nuxt v4: [Nuxt documentation](https://nuxt.com/docs/getting-started/introduction)

## Local

Start the development server on `http://localhost:3000`:

```bash
nvm use
npm install
npm run dev
```

## Deploy

Push to master deploys to production on Netlify.
